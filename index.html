Como o Gartic Obtém Suas Informações

O Gartic tem um mecanismo salvo no navegador que você usa. Mesmo que tenha várias contas no mesmo dispositivo, e mesmo utilizando VPN, ele vai pegar os dados do dispositivo e mesclar com o último IP real que foi usado.

Pessoal do Gartic ainda cai nessa de que só usar VPNs é suficiente para não ser descoberto... Estão muito enganados e eu posso provar.

Como era em 2019

Em 2019, a identificação era feita basicamente pelo IP. Dava para usar VPNs tranquilamente, já que o Gartic não pegava o ID Único do dispositivo.

A partir de 2021

Agora, de 2021 em diante, isso mudou. O sistema do Gartic passou a coletar o ID Único do dispositivo, tornando a identificação muito mais eficaz.

Está aí neste print a relação no sistema deles sobre isso.

Como obtive essas informações?

Quem duplicava antes, mesmo em algo simples como salvar vitórias em uma sala, era facilmente pego pelos banimentos da moderação. Quando alguém era banido, eles enviavam o link do print deles, mostrando o motivo do banimento.

No início de 2024, eu comecei a testar bots nas salas do Geral apenas para entender como eles localizam essas informações. Sacrifiquei contas para isso, mas foi a melhor descoberta que um jogador do Gartic fez até hoje.

Como mudar meu ID Único?

Para alterar o seu ID Único, basta limpar os dados do site Gartic, incluindo os cookies.

É muito simples:

  1. Abra o DevTools (Ferramentas de Desenvolvedor) no seu navegador.
  2. Vá até a categoria "Aplicação" (ou "Application").
  3. Modifique o valor de "aceita" no armazenamento local.
